Detailed Description

The main application class for handling the 3D model viewer.

This class implements the core functionality of the application, including initialization, running the application, handling user input, and managing the window's message loop. It is designed as a singleton to ensure there is only one instance of the application.

Handles error codes and displays appropriate messages.

This function is called to handle various error codes by displaying corresponding error messages to the user.

Parameters
errorCodeThe error code to handle.

Special treatment of selected errors should be implemented here. All errors which need user attention should raise MessageBox() with proper information. All other errors are treated the same simple way, i.e. the user will see "Application error" with error code in the MessageBox.

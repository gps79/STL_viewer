Detailed Description

Represents a 3D model composed of facets.

The CModel class manages the facets that make up the 3D model. It provides methods for manipulating the model's geometry, including normalizing its coordinates and applying rotations around the X, Y, and Z axes. The model is represented as a collection of 3D facets.
